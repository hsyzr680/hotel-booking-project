┌──────────────────────────────────────────────────────────────────┐
│ الخطوة 1️⃣: حدّث ملف .env.local بالرابط الحقيقي              │
└──────────────────────────────────────────────────────────────────┘

  1. افتح ملف .env.local
  
  2. ابحث عن هذا السطر:
     DATABASE_URL="ضع_رابط_Supabase_هنا"
  
  3. غيّره بالرابط الحقيقي من Supabase:
     DATABASE_URL="postgresql://postgres.xxxxx:YOUR-PASSWORD@aws-0-xx.pooler.supabase.com:5432/postgres"
  
  4. احفظ الملف (Ctrl+S)


┌──────────────────────────────────────────────────────────────────┐
│ الخطوة 2️⃣: أضف المتغيرات في Vercel                           │
└──────────────────────────────────────────────────────────────────┘

  افتح: https://vercel.com/dashboard
  
  1. افتح مشروعك (hotel-booking)
  
  2. Settings → Environment Variables
  
  3. أضف الثلاثة متغيرات:
  
     ┌─────────────────────────────────────────────────────────────┐
     │ Key: DATABASE_URL                                           │
     │ Value: <الصق رابط Supabase هنا>                           │
     │ ☑ Production  ☑ Preview  ☑ Development                    │
     │ → Save                                                      │
     └─────────────────────────────────────────────────────────────┘
     
     ┌─────────────────────────────────────────────────────────────┐
     │ Key: NEXTAUTH_SECRET                                        │
     │ Value: 7K9mNpR2tX5wY8zB3cD6fG9hJ2kL5nP8qS1tV4w...          │
     │ ☑ Production  ☑ Preview  ☑ Development                    │
     │ → Save                                                      │
     └─────────────────────────────────────────────────────────────┘
     
     ┌─────────────────────────────────────────────────────────────┐
     │ Key: NEXTAUTH_URL                                           │
     │ Value: https://your-project-name.vercel.app                 │
     │ ☑ Production  ☑ Preview  ☑ Development                    │
     │ → Save                                                      │
     └─────────────────────────────────────────────────────────────┘


┌──────────────────────────────────────────────────────────────────┐
│ الخطوة 3️⃣: انتظر Build وأنشئ الجداول                         │
└──────────────────────────────────────────────────────────────────┘

  في Vercel:
  1. اذهب لـ Deployments
  2. راح تشوف Build جديد يشتغل (بعد ما رفعنا التعديلات)
  3. انتظر 2-3 دقائق لحد ما يخلص
  4. يجب يصير أخضر ✅
  
  بعد ما Build ينجح، افتح PowerShell:
  
  vercel env pull .env.local
  npx prisma db push
  npm run db:seed
